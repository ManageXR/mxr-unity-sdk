# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the ManageXR Unity SDK, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

Instead, please email us at: **security@managexr.com**

We will respond within 48 hours and work with you to understand and address the issue.

## Security Policy

For more information about ManageXR's security practices, see our [Security Policy](https://www.managexr.com/legal/security-policy).
